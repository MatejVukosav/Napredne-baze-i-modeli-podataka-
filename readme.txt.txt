Za izradu web sucelja i backend dijela korišten je visual studio enterprise 2015. Za izradu baze koristen je pgadmin.

Tablice u bazi su:
dani - popis dana u mjesecu
sati - popis sati u danu
texttable - tablica koja sadrzi zapis unesenih podataka (title,body,keywords i summary)
queries - tablica koja sadrzi zapise o pretrazivanjima, sadrzi query tekst, vrijeme i datum izvodenja
